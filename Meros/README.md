Проект: Магазин цемента (демо)
Содержит frontend (public/) и backend (server/)

Как запустить:
1. Установите Node.js (рекомендуется 18+).
2. В корне проекта выполните:
   npm install
3. Запустите сервер:
   npm start
4. Откройте в браузере:
   http://localhost:3000  - магазин
   http://localhost:3000/admin - просмотр заказов

Что сделано:
- Простая SPA-страница (HTML/CSS/JS) с корзиной и формой заказа.
- Сервер на Express, сохраняющий заказы в SQLite (db.sqlite).
- Простая админ-страница для просмотра заказов.

Примечание: это демо — в реальном проекте добавьте валидацию, авторизацию и защиту.
